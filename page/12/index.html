下个月底服务器就要到期了。快过年了，事情太多，我怕给忘了，今天提前在淘宝重新买了一年的服务器，抽空把数据迁移过来了。这次把面板从宝塔换成了1panel。宝塔用久了，总觉太过臃肿，还是 1panel 看着清爽一点。不过，时间过得真快啊，一晃三年就这么过去了。还记得大一那会儿，也是在淘宝，看到服务器三年130块钱的优惠价，我就一口气买了三年。现在回想起来，就跟昨天似的。终究还是没扛住诱惑，剁手了 MACmini M4。天呐噜，入手后只想说简直香爆了！从现在起，它就是我的主力机啦！新年快乐！今年博客上线五周年啦。突然发现，曾经觉得漫长的十年之约，似乎近在咫尺。今天，我无意间瞥到了手机上的到账提醒，催了好久的实习补贴竟到账了。之前，公司信誓旦旦承诺不会私吞我们的钱，我眼巴巴的关注了大半年消息，甚至连其他同学都是从我这儿获取到的人社局公示信息。可结果呢？那些等不及催促的同学早早的拿到了补贴，而我，体谅公司流程繁琐，一直耐心等待，没有刻意去催，却被晾在了一边，仿佛成了软柿子，任人拿捏。不过好在，在我多方沟通后，这笔补贴最终还是给我了，只是这过程，实在让人有些憋屈。给键盘换新衣服了，有点艺术气息了。生日快乐🎂，跟朋友出去吃自助吃的想吐。也是用上佳明540和罗技GPW了。发工资买骑行台了，冬训开始！！！回家乡参加家乡举办的第二场马拉松，去年是家乡第一次举办马拉松也是我人生的第一场马拉松，时间过得好快一晃一年过去了。找到毕业后第一份工作，薪资待遇挺好就是比较累，先干着过度一下吧。

bat批处理脚本一键推送Hexo博客
发表于2021-02-222025-09-19 📔笔记
使用bat批处理的好处：省时间windows中.bat文件可以直接点击运行，我们可以把多个命令写在里面，运行这个.bat文件时就会运行了文件中的所有命令。这样我们运行一次，就等于运行了好多条指令,这会我们省下宝贵的时间。 bat脚本一键推送hexo1234567@echo offD:cd D:\Hexo\Hexogit add -Agit commit -m 'test'git pushpause 指令说明第一条指令@echo off，关闭回显。第二叫指令D:，切换到博客站点目录所在的盘符，我这里是D盘。第三条指令cd D:\Hexo\Hexo，进入站点目录，我这里是D:\Hexo\Hexo。第四条指令git add -A commit -m 'test' git push推送到github。第五条指令pause,暂停。 最后在桌面创建一个名为发布博客.bat的文本文件，粘贴上述代码，然后点击运行，即可发布博客。发布完毕后cmd窗口会自动关闭。

智骑x2max上手体验

丨浅笑安然丨2024-11-162025-09-19

本来在去年上碳刀之前先上骑行台的，但是去年路骑的比较多，骑行台并不是刚需，所以就没有优先购入。今年因为来到上海工作，工作日晚上出去骑车走不远，回来也会比较晚，再加上连续好几个周末下雨和马上快入冬了，最后经过考虑决定购买骑行台。

选购

本来是准备购入智骑刚发布的xx骑行台的，主要看重的是它的价格还有电机直驱和自发电。但是苦于一直没货（万恶的饥饿营销）于是就退而求其次购入了x2换壳的x2max骑行台，主要最近几个月还是有不少朋友购入这款的加上比较低的售价最后还是选择了这款。

因为比较突然周六有雨，周五决定购买之后想要周六就能骑上，如果线上购买我没有安装飞轮的工具，就比较麻烦的。于是经过骑友介绍加上了一个本地有现货的卖家，然后因为骑友也决定购买就跟他一起平摊了一个60元的上门安装费，直接送到骑友那边安装好，我去骑友家取。

最后经过一番周折终于拿到了我的台子，卖家直接帮我组装好了台子并且安装了飞轮。最后售价1269(台子)+120(飞轮)+60(前轮莲花座)+30(上门安装)=1479元

回到家后插电，把自行车装上骑行台。手机安装智骑固件升级工具升级一下骑行台的固件，然后连接码表就可以愉快的骑行啦！当然光连接码表骑是不够的，还需要找一款线上骑行的app增加一些趣味性，这里我选择的是——zwift单机版。

zwift单机版下载及配置

单机版支持在Linux、Windows 和 macOS上运行，因为不同的设备在操作的时候会有一点细微的区别，本文以Windows为主。

  1. 官网下载和安装Zwift

    官网地址：https://www.zwift.com/home 下拉到最下方 Download 选择自己所需的版本

    • Zwift for Windows

    完成安装后，打开Zwift，等待其下载好相关组件，并且登录账号，然后关闭Zwift。

  2. 安装Python环境

    直接Python官网安装https://www.python.org/downloads/

    安装的时候务必勾选And Python 3.xx PATH添加环境变量

    安装完在命令提示行(cmd)输入python回车，显示如下内容就成功了。

  3. 安装zwift-offline

    打开https://github.com/zoffline/zwift-offline，在右边Releases下载最新版。

    • 一键直达

    这里我们下载后缀.exe的可执行文件就行，如：zoffline_1.0.136426.exe
    然后新建一个文件夹将.exe文件放进去，建议和Zwift放在相同的目录里方便管理。注意新建的文件夹里不要有其他内容！

    然后双击文件夹里的zoffline_1.0.136426.exe文件，之后弹出运行的窗口了，并且在当前文件夹下生成了配置文件，这样我们单机版就基本安装完成了。

  4. 写入本地Hosts

    管理员权限打开C:\Windows\System32\drivers\etc\Hosts文件在地下新增一行127.0.0.1 us-or-rly101.zwift.com secure.zwift.com cdn.zwift.com launcher.zwift.com然后保存

    如果你想使用在线版，后期只需要在Hosts文件中注释掉这一行，保存即可（使用记事本打开，注释在前面加个#，Ctrl+s保存）

  5. 运行zwift单机版

    每次运行前要先打开zwift-offline，也就是前面下载的exe文件，在确保弹出窗口运行后(不能关闭)再打开桌面上的zwift，然后耐心等待一下加载就可以进入单机版zwift开始愉快的骑车了。

  6. 进阶

    • 添加机器人：打开https://github.com/oldnapalm/zoffline-bots 下载bots.zip并且解压。然后移动到zwift-offline的根目录下，其中一个文件夹和根目录下配置文件名称相同，直接覆盖即可。这样在骑车的时候就会自动生成一些机器人，营造一种很热闹的感觉，增加骑行乐趣。

    • 记录上传到Zwift：打开https://github.com/oldnapalm/zoffline-helper下载[zoffline-helper.zip](https://github.com/oldnapalm/zoffline-helper/releases/download/zoffline-helper/zoffline-helper.zip)并解压。

      在 Windows Firewall 中运行并允许它。strava_auth.exe –client-id CLIENT_ID –client-secret CLIENT_SECRET
      打开 http://localhost:8000/ 并授权。
      将结果移动到配置文件storage\1目录中。storage\1\strava_token.txt

    • 记录上传到Garmin Connect国区

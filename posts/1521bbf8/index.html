实用软件推荐

发表于2021-09-24|更新于2023-04-15|📦干货分享

不知不觉建站快两年了，在折腾的过程中遇到了很多问题，也接触到了很多实用的软件。于是就有了这篇文章，用于记录我在日常使用电脑以及建站过程中使用的一些软件。

日常类

火绒安全

火绒安全是一款小巧的安全防护软件，虽然系统有自带防护软件，但是火绒的一些自带工具还是很实用的，并且可以强制弹出移动设备，解决了弹出占用问题。

geek

geek是一款小巧实用的软件卸载工具，卸载完会清理注册表等残留，并且可以强制删除。

7-zip

7-zip是一款高压缩比的压缩软件，不仅支持独有的7z文件格式，而且还支持各种其它压缩文件格式，其中包括 ZIP、RAR、CAB、GZIP、BZIP2和TAR等格式。此软件压缩的压缩比要比普通ZIP文件高30-50%。

PyDebloatX

PyDebloatX是一款Win10预装应用删除工具，可以一键删除win10自带的一些软件。

Wub

Wub是一款Win10自动更新关闭工具，可以一键关闭烦人的更新。

建站类

Everything

Everything是voidtools开发的一款文件搜索工具，Everything有极快的搜索速度,运行时占用的内存低。

Visual Studio Code

Visual Studio Code是微软公司向开发者们提供的一款跨平台编辑器，功能强大而且免费，是写代码的不二之选。

Typora

Typora是一款极简免费的Markdown编辑器，软件支持markdown的标准语法，还支持动态预览功能，一键预览，让一切都变得如此干净、纯粹，是一款不可多得的优质markdown编辑器。

PicGo

PicGo是一个开源免费的图片上传 + 管理工具，支持 macOS、Windows 和 Linux 系统，配合我们的写作工具Typora，可以很方便地将图片上传到图床上。

Imagine

Imagine是一款实用的图片压缩工具，让我们以质量的微小损失换取大量的存储空间和更快的访问速度。

SwitchHosts

SwitchHosts是一款管理 hosts 文件的利器，配合GitHub520即可解决Github打不开的问题。
